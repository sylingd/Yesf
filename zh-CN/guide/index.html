# Yesf

Yesf是基于Swoole 4.0+的框架。具有以下优点：

  • 高性能
  • 灵活、扩展能力强
  • 单元测试覆盖

Yesf基于Swoole，因此还支持TCP监听、UDP监听、异步任务等功能

# DEMO

此处有一个简单的demo：sylingd/Yesf-Example，您可以结合demo了解如何使用Yesf

# 文档说明

本文档对应Yesf版本为2.0.0，如有错误请提交issue至GitHub或Gitee

# PSR规范

目前遵循以下PSR规范：

  • PSR-1: Basic Coding Standard
  • PSR-3: Logger Interface
  • PSR-4: Autoloader
  • PSR-11: Container interface
  • PSR-16: Simple Cache

# 功能说明

部分功能因为和框架本身关系不大，且已经有很多优秀的第三方库，因此Yesf不再内置它们。建议直接使用composer安装，例如：

  • 输入验证：respect/validation nette/utils
  • 图片验证码：gregwar/captcha dapphp/securimage
  • 图片处理：intervention/image nette/utils
  • JWT：firebase/php-jwt lcobucci/jwt
