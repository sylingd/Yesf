# 快速开始

## 环境需求

* Swoole 4.0+
* PHP 7.1+
* Composer

## 可选功能

* 日志功能建议安装[SeasLog扩展](https://seasx.github.io/SeasLog/)
* 热重载（仅开发模式存在）需要[inotify扩展](https://pecl.php.net/package/inotify)

## 安装Swoole

参见[Swoole官方文档](https://github.com/swoole/swoole-src/blob/master/README.md)
