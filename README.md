Yesf是一个基于Swoole 4.0+的PHP框架，具有高性能、简单方便、可靠、易扩展的优点。提供了以下功能：

* IoC/DI
* 完整的路由
* MVC支持
* 协程连接池
* 多端口监听

Yesf is a PHP framework based on Swoole 4.0+. The following functions are provided:

* IoC/DI
* Full router
* MVC mode
* Multi-port monitoring
* Coroutine connection pool
